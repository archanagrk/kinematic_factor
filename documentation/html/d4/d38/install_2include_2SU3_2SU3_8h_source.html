10 /* Macros to iterate over the states of a given coupling. We guarantee that:
11  * All states produced are valid
12  * All of the states with nonzero coupling are produced
13  But we do *not* guarantee which states of zero coupling are produced,
14  or in what order.
15  Additionally, we do not iterate over degenerate representations.
16  Each argument is evaluated multiple times, and k*, l* are expected to
17  be simple variables.
18 
19  Currently, we use the following properties to reduce the number of states
20  iterated over:
21  i) Hypercharge conservation implies
22  k1 + l1 + k2 + l2 - k - l = (2*p1 + 2*p2 + 4*q1 + 4*q2 - 2*p - 4*q)/3
23  ii) Isospin conservation implies that:
24 
25  |i1 - i2| <= I <= i1 + i2
26  i1 + i2 - I is an integer
27  Iz = i1z + i2z
28 
29  or, in terms of our variables, where I = (k-l)/2, Iz = (2*m-k-l)/2,
30 
31  |k1 - l1 - k2 + l2| <= k - l <= k1 - l1 + k2 - l2
32  k1 - l1 + k2 - l2 - k + l is even
33  2*m - k - l = 2*m1 - k1 - l1 + 2*m2 - k2 - l2
34 
35  where we can rewrite the last condition as
36  m2 = (m - m1) - (k + l - k1 - l1 - k2 - l2)/2
37 */

40  for (l = 0; l <= q; ++l) \
41  for (k1 = q1; k1 <= p1+q1; ++k1) \
42  for (l1 = 0; l1 <= q1; ++l1) \
43  for (k2 = q2; k2 <= p2+q2; ++k2) \
44  /* Assign l2, then check it is in range */ \
