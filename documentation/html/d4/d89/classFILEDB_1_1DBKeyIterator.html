Detailed Description

template<typename T, typename D>
class FILEDB::DBKeyIterator< T, D >

Template class of DB iterator Template class T is key for the database. T should be derived from DBKey Class. T should have = and == operators defined

Template class D is data for the database. D should be derived from DBData Class

The parameter T is what returned by iterator
