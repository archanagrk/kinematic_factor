◆ SPLITSHIFT

#define SPLITSHIFT   11

Overflow page numbers are allocated per split point. At each doubling of the table, we can allocate extra pages. So, an overflow page number has the top 5 bits indicate which split point and the lower 11 bits indicate which page at that split point is indicated (pages within split points are numberered starting with 1).
