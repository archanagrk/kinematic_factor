Detailed Description

Interface (abstract class) for multi-dimensional functions providing a gradient calculation. It implements both the ROOT::Math::IBaseFunctionMultiDim and

ROOT::Math::IGradientMultiDim interfaces. The method ROOT::Math::IFunction::Gradient calculates the full gradient vector, ROOT::Math::IFunction::Derivative calculates the partial derivative for each coordinate and ROOT::Math::Fdf calculates the gradient and the function value at the same time. The pure private virtual method DoDerivative() must be implemented by the derived classes, while Gradient and FdF are by default implemented using DoDerivative, butthey can be overloaded by the derived classes to improve the efficiency in the derivative calculation.
