PSeed<T> = PSeed<T> * PSeed<T>

A 47 bit seed multiplication is represented as the multiplication of three 12bit and one 11bit integer

i3 = s1(3)*s2(0) + s1(2)*s2(1)

  • s1(1)*s2(2) + s1(0)*s2(3); i2 = s1(2)*s2(0) + s1(1)*s2(1)
  • s1(0)*s2(2); i1 = s1(1)*s2(0) + s1(0)*s2(1); i0 = s1(0)*s2(0);

dest(0) = mod(i0, 4096); i1 = i1 + i0/4096; dest(1) = mod(i1, 4096); i2 = i2 + i1/4096; dest(2) = mod(i2, 4096); i3 = i3 + i2/4096 dest(3) = mod(i3, 2048);
